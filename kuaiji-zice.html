2 · ROA 与利润驱动分解

平均总资产 = (286,556 + 301,311) / 2 = 293,933.5 ROA = 44,281 / 293,933.5 ≈ 0.15（15%） 净利率 Net Income/Net Sales = 44,281 / 143,015 = 0.31 ×周转率 Net Sales/Avg Total Assets = 143,015 / 293,933.5 = 0.49 = ROA = 0.31 × 0.49 = 0.15

微软净利率高（每 $1 销售赚 $0.31），资产效率每 $1 资产产生 $0.49 销售；其软件业的主导地位使其能收取溢价。

3 · 额外 $1,500 G&A（现金）的影响

2 · 为什么每月调节

每月调节能确保账簿反映正确余额、及时发现错误、避免开出空头支票。企业还有额外原因：调节得出的正确余额需列报于资产负债表。

FIFO / LIFO · 存货周转

题目 · Balent Appliances（微波炉）

截至 2022 年 12 月 31 日，按顺序发生（均为现金）：

资料
a. 期初存货 Beginning inventory（11 件）$200/件
b. 新采购 New purchases（9 件）$220/件
c. 销售 Sales（8 件，售价 $420）?
要求 Required
  1. 分别用 FIFO、LIFO 计算期末存货（件 / 金额）与销售成本（件 / 金额）。
  2. 若成本预计延续当前趋势，建议用哪种方法？解释。
  3. 其他营业费用 $500、税率 25%，用所选方法编制损益表。
  4. 用所选方法计算存货周转率，并说明含义。
Suggested Solution
关键：可供销售 $4,180（20 件）；FIFO 期末 $2,580 / COGS $1,600；LIFO 期末 $2,420 / COGS $1,760。涨价应选 LIFO；LIFO 下净利 $825，存货周转 0.76。

1 · FIFO / LIFO

可供销售 = 11×200 + 9×220 = 2,200 + 1,980 = 4,180（20 件）；售 8 件，余 12 件 FIFO 期末（最近成本）= 9×220 + 3×200 = 2,580 → COGS = 8×200 = 1,600 LIFO 期末（最早成本）= 11×200 + 1×220 = 2,420 → COGS = 8×220 = 1,760
期末存货 / 销售成本
FIFO（12 件 / 8 件）$2,580 / $1,600
LIFO（12 件 / 8 件）$2,420 / $1,760

2 · 方法选择

应选 LIFO。成本上升时，LIFO 产生更高 COGS、更低税前利润、更少所得税；依 LIFO 一致性规则，税表与损益表须同时采用。

4 · 存货周转率

存货周转 = COGS / 平均存货 = 1,760 / [(2,200 + 2,420)/2 = 2,310] = 0.76

它反映期内平均存货被采购并售出的次数。Balent 全年平均存货周转不足 1 次。
